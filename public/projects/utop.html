UTop Tutor

Started: September 2016
Latest update: April 2017

Tags: Node.js, Express, MongoDB, HTML/CSS, Web App
How: Visit this Link
Objective: Designed for UTop Tutor, an organization that offers tutoring services for students in UofT. Users can log in and make appointments with tutors.

The website for UTop Tutor, a student organization.

  • Phase 1: September 2016 - December 2016
    • The website is built with Apache web server.
    • The backend is written in PHP.
    • MySQL is used for storing user and order information.
    • When an order is placed, an automatic email is sent to the user by PHP Mailer.
    • This old version of website can be reached here or on GitHub but is not functional
  • Phase 2: January 2017 - March 2017
    • New feature added: auto-tester for students' assignment.
    • Students can upload their python file for CSC108 assignment and will be replied with a score. One IP address has three chances per day.
    • This feature is naively implemented in PHP at first
      Some students' code contains infinite loop and the auto-tester will run it forever.
      Thus, a "kill" command was added to terminate timeout processes.
    • As it becomes more popular, I noticed that some saboteur uploaded a fork bomb and attemped to stole files on the server on seperate days.
      The fork bomb crisis was solved by disabling the sys module in the Python test file.
    • Statistics shows that the auto-tester was ran over 3000 times during a two-week period.
  • Phase 3: April 2017 - May 2017
    • A new website is built with Node.js Express server and the backend is of course written in JavaScript.
    • This is a RESTful web application. The user and the server communicates via GET and POST requests.
    • The frontend web pages are rendered dynamically and Bootstrap is used for CSS.
    • NoSQL database MongoDB is chosen for this app.
    • Users are able to update their personal information.
    • For better extensibility, NGINX proxy server was added to direct visits from different URL to according port.
