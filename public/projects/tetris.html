My Tetris

Started: November 2013 Finished: December 2013

Tags: C, Microsoft Virtual C++ 6.0, Game Design
How: Download from Github and run release/方块拼图V2.2.1224.exe in Windows XP/Vista/7/8.x.
Objective: A game similar to Tetris.

By the time I implemented this game, I only learnt how to write code in C, but had no knowledge of coding style. Thus, although this game had been a great achievement for me in 2013, it is actually terriblely written.

Rules:

  • Use mouse or keyboard to move and place tetrominoes onto the board.
  • When there is a 3 by 3 or larger region that is all placed with tetrominoes, they are all destroyed and the player gains scores.
  • The player comes to the next stage if they place enough tetrominoes, but the tetrominoes remained when level-up can become undestroable with probability.
  • The player can buy tool by spending scores they earned to help them.
